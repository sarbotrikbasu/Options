# Privacy Policy

**Effective Date:** March 10, 2026

## 1. Introduction

This Privacy Policy describes how the **NSE Options Analytics API** (“the Service”) handles information when users access or interact with the API hosted at:

https://options-x4dj.onrender.com

The Service provides analytics related to Indian derivatives markets such as option chain summaries, put–call ratio calculations, support/resistance levels, max pain, and option Greeks.

The Service is designed to operate with minimal user data collection.

---

## 2. Information We Collect

### 2.1 Information Provided by Users

The API accepts limited input parameters required to perform analytics calculations. These parameters may include:

* Market symbol (e.g., NIFTY)
* Option strike price
* Option type (CE or PE)

These inputs are used only for computation and are not stored permanently.

### 2.2 Automatically Collected Information

When accessing the Service, certain technical information may be automatically collected by the hosting platform or infrastructure providers. This may include:

* IP address
* Request timestamps
* HTTP headers
* Usage logs
* Error logs

This information is used strictly for system monitoring, security, and service reliability.

---

## 3. How We Use Information

Information received by the Service is used solely for the following purposes:

* Processing API requests
* Generating options analytics calculations
* Monitoring system performance and reliability
* Detecting abuse or security issues

The Service does **not use collected information for advertising or profiling**.

---

## 4. Data Storage and Retention

The Service does not intentionally store user input or personal information in a database.

Temporary logs may be maintained by the hosting infrastructure for operational purposes. These logs may be retained for a limited period depending on the policies of the hosting provider.

---

## 5. Third-Party Services

The Service relies on third-party infrastructure providers to operate, including:

* Cloud hosting services
* Market data providers
* API infrastructure services

These providers may process technical data necessary for operating the Service according to their own privacy policies.

## 6. Data Security

Reasonable technical and operational safeguards are used to protect the Service and its infrastructure. However, no internet transmission or storage system can be guaranteed to be completely secure.

Users should avoid submitting sensitive personal information to the API.
